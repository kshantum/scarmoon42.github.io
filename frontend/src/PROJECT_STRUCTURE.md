# Структура проекта - Система управления кадровой политикой СибГИУ

## Обзор
Информационная система для управления кадровой политикой Сибирского государственного индустриального университета (СибГИУ).

## Новые возможности ✨

### Выход из аккаунта
- ✅ Кнопка "Выйти" доступна на **всех страницах** после авторизации
- ✅ Пользователь может выйти из системы в любой момент
- ✅ При выходе происходит сброс состояния (выбранный кандидат, текущая страница)
- ✅ После выхода пользователь возвращается на страницу входа

### Расположение кнопки выхода:
- Преподаватель: на всех страницах (выбор роли, главная, загрузка файлов, тестирование)
- Эксперт: на всех страницах (выбор кандидата, главная, документы, чек-листы, анкета, открытое занятие)
- Секретарь: на всех страницах (главная, управление пользователями, назначение занятий, рейтинг)

## Роли пользователей

### 1. Преподаватель (teacher / 1234)
- Выбор должности и кафедры
- Загрузка УМК, КП и документов о ПК
- Прохождение тестирования
- Доступ к QR-коду для студентов

### 2. Эксперт (expert / 1234)
- Выбор кандидата для оценки
- Просмотр документов кандидата
- Заполнение чек-листов
- Проведение анкетирования
- Оценка открытого занятия

### 3. Секретарь (secretary / 1234)
- Управление пользователями (создание, удаление, назначение ролей)
- Назначение открытых занятий
- Просмотр рейтинга преподавателей
- Работа с временными аккаунтами

### 4. Студент (анонимный доступ через QR-код)
- Оценка преподавателя по чек-листу открытого занятия

## Ключевые особенности

### Временные аккаунты
- Преподаватели и внешние эксперты могут иметь временные аккаунты
- Секретарь устанавливает дату удаления при создании аккаунта
- Аккаунт автоматически удаляется по достижении указанной даты

### QR-коды
- Генерируются для каждого открытого занятия
- Студенты получают анонимный доступ к чек-листу
- Преподаватель может показать QR-код во время занятия

### Система оценки
- Студенты: 0-3 балла по 5 критериям
- Эксперты: чек-листы, анкеты, оценка открытого занятия
- Итоговый рейтинг: автоматический расчет на основе всех оценок

## Технологии
- React + TypeScript
- Tailwind CSS v4.0
- Radix UI для компонентов
- Lucide React для иконок

## Навигация в приложении

```
Login → {teacher/expert/secretary}

Teacher Flow:
Login → Role Selection → Teacher Home → {Upload UMK, Upload PK, Testing, QR Code}

Expert Flow:
Login → Candidate Selection → Expert Home → {Documents, Checklists, Survey, Open Lesson}

Secretary Flow:
Login → Secretary Home → {User List, Assign Lesson, Rating}

Student Flow:
QR Code → Student Checklist (anonymous)
```
